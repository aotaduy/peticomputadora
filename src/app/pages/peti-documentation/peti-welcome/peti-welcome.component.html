Bienvenidos

Este es un simulador de una computadora minima didactica llamada PetiComputadora

Es una computadora con una tamaño de palabra de 8 bits, 3 bits para el codigo de instruccion y 5 bits ´para el direccionamiento de memoria.

Lo que redunda en 8 instrucciones y 32 posiciones de memoria disponibles de 8 bits cada una.

En la pagina de Ejemplos se pueden ver varios programas de ejemplo para ejecutar.

Sino en la pagina del emulador propiamente dicho podemos ejecutar cualquier programa
